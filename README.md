# Sistema Web Institucional – APAE de Tijucas

## 1. Apresentação do Projeto

Este projeto foi desenvolvido no contexto do **Projeto Integrador Extensionista**, da disciplina de Responsive Web Developer.

A proposta consiste no desenvolvimento de um sistema web institucional voltado à divulgação e apoio às atividades da APAE de Tijucas, com foco na promoção da causa, organização de informações e incentivo à participação da comunidade.

O sistema foi desenvolvido com o objetivo de oferecer uma solução simples, acessível, responsiva e de baixo custo de manutenção, possibilitando que a própria instituição possa utilizá-lo e gerenciá-lo futuramente.

---

## 2. Sobre a Instituição

A APAE de Tijucas é uma Organização Não Governamental (ONG) sem fins lucrativos que atua no atendimento a pessoas com deficiência intelectual e múltipla.

A instituição oferece suporte educacional, terapêutico e social, promovendo inclusão, desenvolvimento e melhoria da qualidade de vida dos assistidos, além de apoiar suas famílias.

Seu trabalho é essencial para o fortalecimento da inclusão social no município e região.

---

## 3. Objetivo do Sistema

O sistema web foi desenvolvido com a finalidade de:

- Divulgar a instituição e sua atuação;
- Apresentar eventos promovidos;
- Disponibilizar canal para doações;
- Organizar informações institucionais;
- Facilitar o engajamento da comunidade.
