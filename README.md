# 🌐 Projeto Integrador – Sistema Web APAE Tijucas

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina de **Responsive Web Developer**, no Projeto Integrador Extensionista.

O sistema foi criado para a **APAE de Tijucas**, com o objetivo de melhorar a divulgação institucional, facilitar o engajamento da comunidade e organizar informações como eventos e doações.

O foco principal é oferecer uma solução digital simples, responsiva e de baixo custo de manutenção.

---

## 🎯 Objetivo

Desenvolver um sistema WEB responsivo que permita:

- Divulgação da instituição
- Apresentação de eventos
- Recebimento de doações
- Área administrativa para gerenciamento de conteúdo
- Página institucional explicando a causa

---

## 🏢 Entidade Atendida

**Nome:** APAE de Tijucas  
**Tipo:** Organização Não Governamental (ONG)  
**Área de atuação:** Atendimento a pessoas com deficiência intelectual e múltipla  

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Design Responsivo  
- GitHub para versionamento  

---

## 📱 Estrutura do Sistema

O sistema é composto pelas seguintes páginas:

### 🏠 Página Inicial (index.html)
- Apresentação da instituição
- Informações institucionais
- Navegação principal

### 📅 Eventos (eventos.html)
- Divulgação de eventos
- Organização de atividades promovidas pela instituição

### 💙 Doações (doacoes.html)
- Formulário para contribuição
- Incentivo ao apoio da causa

### 🔐 Login (login.html)
- Acesso à área administrativa

### ⚙ Painel Administrativo
- Gerenciamento de eventos
- Organização de informações

### 📖 Sobre (sobre.html)
- História da instituição
- Missão, visão e valores

---

## 📊 Justificativa do Layout

O layout foi desenvolvido com foco em:

- Interface simples e intuitiva
- Cores institucionais
- Fácil navegação
- Responsividade para dispositivos móveis
- Baixo custo de manutenção
